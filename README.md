# 🏏 Cricket Turf Booking System

A full-stack web application that allows users to book cricket turfs online with ease. Users can browse available turfs, check availability, book slots, and manage their reservations. The application provides a simple and user-friendly interface for both customers and administrators.

---

## 📌 Features

- 👤 User Registration & Login
- 🏏 Browse Available Cricket Turfs
- 📅 Book Turf Slots
- ⏰ Real-Time Slot Availability
- ❌ Cancel Bookings
- 👨‍💼 Admin Dashboard
- 🏟️ Add, Edit & Remove Turfs
- 📱 Responsive Design

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/cricket-turf-booking.git
```

### Navigate to the Project

```bash
cd cricket-turf-booking
```

### Install Dependencies

```bash
npm install
```

### Start the Application

```bash
node app.js
```

or

```bash
npm start
```
